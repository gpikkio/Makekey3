                                                                              
                                                                              
 GLONASS SPK Files                                                               
  =============                                                               
                                                                              
  This directory contains:

  bodydef_glonasspr.ker             SPICE definition of GLONASS satellite

  prNN_YYDDMM_vYYDDMM.bsp           SPK orbit kernel of each satellite. NN is the 
                                    number of the satellite, first YYDDMM is the date of the data, 
                                    and vYYDDMM is the file release date.
